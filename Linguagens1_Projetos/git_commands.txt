Usando git bash
cd -> change directory
git clone -> usar e colocar o URL do site GIThub
cd ..-> volta para o diretório anterior
git add . -> adiciona ao que será puxado tudo que foi recém adicionado ou que teve novas mudanças
git commit "nome da acao" -> envia para o sistema
git status -> autoexplicativo

javac nomearquivo.java -> compila código
java nomearquivo -> run code

git config --global user.email "email do git"
git config --global user.name "username do git"
os git global são só se necessários

git remote add origin site_git (se necessário)

git push -u origin master -> envia para o GitHub
